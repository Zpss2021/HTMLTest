块元素:

代表标签：div p form table h1 hr ol ul pre

1、元素独占一行
【父元素】
2、默认宽度与父元素相同、高度由内容撑开
3、可以设置宽高和内外边距
4、可容纳块元素、行内元素、行内块元素和文本等

行内元素:

代表元素：span a b i br img cite code

1、元素可在一行 2、遇父级元素边界换行，宽高由内容撑开
3、不可以设置宽高和内外边距 4、仅能容纳行内元素、行内块元素或文本

行内块元素：

代表标签：input textarea button select img(有其特点)

【父元素】
div转行内元素1
span转行内块元素
div转行内元素2
span转块元素
